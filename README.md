# 💼 ProHire – Job Portal Website

![ProHire Banner]([https://via.placeholder.com/1200x400?text=ProHire+Job+Portal](https://opensourcecollection.com/projects-storage/images/micropyramid-opensource-job-portal-2.png))

## 🚀 Overview
**ProHire** is a full-stack job portal web application that connects job seekers with recruiters.  
Users can browse jobs, apply for positions, and manage applications, while recruiters can post jobs and manage candidates.

---

## ✨ Features

### 👨‍💻 Job Seekers
- 🔍 Search & filter jobs
- 📄 Apply to jobs
- 📝 Create & manage profile
- 📌 Save favorite jobs

### 🏢 Recruiters
- ➕ Post new jobs
- ✏️ Edit/Delete job listings
- 👀 View applicants
- 📊 Manage job postings

### 🔐 Authentication
- Secure login/signup
- Role-based access (User / Recruiter)
